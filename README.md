# UseMart
UseMart is an innovative webbases e-commerce platform dedicated to the 
buying and selling of second-hand goods. The platform aims to promote 
sustainability by reducing waste and encouraging the reuse of items. 
It provides a secure, userfriendly environment for individuals to find 
high-quality preowned products, ranging from electronics and clothing 
to furniture and books. 

UseMart's comprehensive features ensure a seamless and trustworthy 
experience for both buyers and sellers 

# Project Technology
Front End: HTML, CSS, & JavaScript  
Back End: Python & PHP 
Database: MySQL 
Server: Apache

# Software Used: 
Figma, Draw.io, VsCode, XAMPP.
